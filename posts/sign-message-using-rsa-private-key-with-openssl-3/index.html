Prerequisites#

  • OpenSSL 3.0.8 built/installed
    • [How to build and install OpenSSL 3.0.8 on linux]
    • Why OpenSSL 3.0.8?
      • OpenSSL 3.0.8 is the FIPS validated version (Certificate)(Valid until: 9/21/2026)
  • OpenSSL 3 headers, OpenSSL 3 crypto library (libcrypto.a), OpenSSL 3 openssl.cnf and OpenSSL 3 fipsmodule.cnf
    $ mkdir -p ~/projects/rsa-private-key-sign/
$ cd ~/projects/rsa-private-key-sign/
$ cp -r ~/installed/openssl-3.0.8/{ssl,include,lib} ~/projects/rsa-private-key-sign/

$ ls -gG ~/projects/rsa-private-key-sign/

Generate an RSA private key (programmatically and using openssl command)#

  • Use openssl binary to generate a sample 8192 bit RSA private key and public key pair
$ openssl genpkey -quiet -algorithm rsa -pkeyopt rsa_keygen_bits:4096 -out id_rsa
$ head -1 id_rsa
-----BEGIN PRIVATE KEY-----

$ openssl rsa -in id_rsa -pubout -out id_rsa.pub
writing RSA key

$ head -1 id_rsa.pub
-----BEGIN PUBLIC KEY-----
